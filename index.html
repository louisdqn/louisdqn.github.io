Natural Language Property Search

+35% lead qualification

A chatbot where users type things like "villa in Moraira with pool under 800k" and get matching results. Built on a RAG pipeline: Claude extracts filters, embeddings go through Pinecone for semantic search, then Claude writes a conversational response. Bilingual, with follow-up support ("show me cheaper ones").

Next.js FastAPI Pinecone Claude Python

AI Inside Sales Agent

Goes beyond search. This agent qualifies leads, analyzes property photos with Claude's vision, captures contact info and preferences from the conversation, and books viewings by managing calendar slots. Intent detection routes the conversation so it feels natural, not scripted.

Python ChromaDB Claude Vision FastAPI PostgreSQL Vercel

Energy Bill Savings Agents

470+ bills analysed

A multi-agent system that helps users save on their electricity bills. One agent extracts key info (name, address, PDL number) from uploaded bills using Mistral OCR. A second one pulls real consumption data from the Enedis API. A third analyzes everything - bills, contract, provider - and gives personalized recommendations to cut costs.

Claude Mistral OCR Enedis API Python FastAPI Next.js
